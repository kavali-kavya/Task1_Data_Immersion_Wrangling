# Task 1 - Data Immersion & Wrangling

## Objective

The objective of this task was to understand, clean, and prepare a sales dataset for analysis.

## Dataset Overview

The dataset contains customer information, product details, order information, and sales data.

## Data Profiling

The dataset was analyzed to identify:

* Missing values
* Duplicate records
* Data types

## Data Cleaning

The following cleaning steps were performed:

* Filled missing Age values using the median.
* Filled missing City values with "Unknown".
* Removed duplicate records.
* Converted Order_Date into datetime format.

## Feature Engineering

New columns were created:

* Month
* Year
* Age_Group
